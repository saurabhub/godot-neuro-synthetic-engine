# 🧠 Godot Neuro-Synthetic Data Engine (v1.0)

A professional simulation environment built in **Godot 4.6** for generating synthetic datasets for **LLM Reasoning** and **Reinforcement Learning**.

## 🚀 Key Capabilities
- **Advanced RayCast Vision:** AI agents perceive environment geometry in 360°.
- **Decision Metadata Logging:** Captures not just position, but the *logic* behind every move.
- **Research-Ready Exports:** Generates structured JSON data for Python/Pandas analysis.

## 🛠 Setup
1. Download Godot 4.6 Standard.
2. Clone this repo: `git clone https://github.com/YOUR_USERNAME/godot-neuro-synthetic-engine.git`
3. Import `project.godot` into the engine.
4. Press **F5** to start the simulation.

## 📊 Output
Data is saved to `user://simulation_data.json` containing timestamps, vectors, and state-machine transitions.
